Gradient separator mocks

Each tile is the page colour up top fading into the cream Find-us surface below. Pick a source colour, then compare how each variant releases the colour — the goal is no hard band where the gradient starts and the smoothest possible blend into cream.

The old separator started with a hard band because the gradient held the source colour for the first ~third ({from} 32%, cream) — so the top of the fade read as a darker stripe before any blend began. All variants below drop that hold.

V3 (oklab, eased) is the one wired into the app right now. To try a different one, edit .findus-fade in src/ui/components/FindUs/FindUs.css — each variant's exact CSS is shown under its tile (open this page's source, or read the labels).
